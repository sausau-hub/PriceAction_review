## 二、Globex 夜盘关键观察（Al Brooks）

**限价单多头连续盈利** = 空头趋势不强的信号：
```
每次创新低 → 市场反弹回该低点以上
→ 限价单多头：在低点买入，再低5点加仓，最终获利
→ 这种情况在强空头趋势里不应该发生
→ 结论：空头趋势正在弱化，可能转入区间
```

**Globex 熊通道统计规律：**
- 75% 概率：从通道上方突破（多头方向）
- 25% 概率：熊侧突破并量度下跌

今日结果：**向下突破 Globex 低点**，但随即反弹 = 假突破 + 限价单多头接盘

## 六、本场核心知识点

### 1. 远离均线时的做空规则
```
价格远低于 20棒 EMA → 不应轻易追空
条件：必须看到连续大阴棒 + 收盘接近低位 + 破前低
否则：多头在更低位置有买盘，止损太宽，数学不合理
```

### 2. Globex 熊通道的两种结局
```
75% → 上方多头突破（转区间或多头）
25% → 向下熊侧突破（量度下跌）

今日 = 25% 的情况，但：
  - 熊侧突破后反弹（Globex 低点买盘）
  - 限价单多头盈利 → 空头不强
  - 最终走 60% 概率：横盘区间
```

### 3. 限价单多头判断空头趋势强度
```
规律：在空头趋势中，如果限价单多头持续盈利
→ 说明每个新低都有真实买盘
→ 空头趋势正在弱化
→ 可能转区间（而不是崩跌）

应用：判断今日是否真正会崩跌的关键过滤条件
```

### 4. II 形态（双内部棒）在趋势末尾的含义
```
在空头趋势尾部出现 II：
① 向上突破 → 直接反转
② 向下突破 → 最后一个空头旗形，仍然是反转的前兆

无论哪种 → 都是非常高概率的短期反转信号
操作：在 II 以上设停止单买入（或至少平空）
```

### 5. 60分钟大棒后的"测试一两个tick"规律
```
大阴棒出现后：
  → 市场常会在下一根或两根棒测试该阴棒低点以下1-2tick
  → 然后反转（假突破）

大阳棒出现后：
  → 同理测试高点以上1-2tick后反转

这不是真正的突破，是机器测试买卖力量的机制
应用：在 60分钟大棒附近，看到微小突破不要立即追
```

### 6. 卖出高潮后的三种结局及概率（Al Brooks 标准）
```
大幅低开 + 空头趋势开盘：
  ① 60% → 停止下跌，横盘至少2小时（本日结局）
  ② 20% → 全天空头趋势日，持续下跌
  ③ 20% → 反转，强多头趋势日

卖出高潮后反转：
  → 第一目标 = 最近一段卖出高潮的顶部
  → 70% 概率先来一次小反弹，然后还有第二段测试
  → 最终 30% 机会发展为真正多头趋势
```

### 7. Rose 的棒体收盘（Body Close）优先原则
```
Rose 排除影线，只看棒体：
  → 机器大量使用棒体收盘价设定支撑/阻力/进场点
  → 棒体 Gap（相邻棒之间没有重叠的棒体区域）= 真正的被套区域
  → 需要回测棒体 Gap 才能"解救"被困的交易者

实例：K28-K30 棒体 Gap → 市场最终会回测该区域
```

### 8. 量度下跌目标的层次（Rose 方法）
```
从旗杆量度（以最后一根有效空头裸棒的 close 为起点）：
  1R = 移动约等于旗杆高度（空头场景）
  2R = 旗杆高度 × 2
  3R = 旗杆高度 × 3

本日计算：
  从周四收盘 → 今日开盘 close（旗杆）
  1R ≈ 已在开盘后快速达到
  2R ≈ 5875 区域（约午前达到）
  3R ≈ 延伸目标（未必当天到达）
```

### 9. 均线磁吸效应（Al Brooks）
```
Al 的统计（40年交易经验）：
  99.9% 的交易日，价格会在日内触碰 20棒 EMA 至少一次
  仅见过一次全天未触及均线的情况

应用：
  今日开盘距均线 90+ 点 → 必然会触及均线
  → 任何反弹的第一目标就是均线
  → 为多头回补空头提供了理由
```

### 10. 大整数与量度目标的叠加效应
```
5200 = 大整数（心理关口）
5200 ≈ Globex 量度下跌目标
5200 ≈ 60分钟图量度下跌目标
→ 三重叠加 → 强力支撑

结果：价格在 5200 精确触底（计算机精确到 tick 执行）
```
