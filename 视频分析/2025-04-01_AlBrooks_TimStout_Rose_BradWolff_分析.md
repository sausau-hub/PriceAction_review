### ▌ATR 6点阈值规则（Rose 独创讲解）

Rose 详细解释了她核心的"6点阈值"概念：

| ATR状态 | 标准偏差（Scalp） | 摆动目标（Swing） | 意义 |
|---------|-----------------|-----------------|------|
| ATR 3-5点（历史正常） | 5点 | 10点 | 疫情前标准 |
| ATR 8-10点（当前市场） | **10点** | **20点** | 疫情后新标准 |
| 机器判断信号 | **收盘体超过6点** | → 预示再走10点 | 6点 = 5点scalp + 1点机器信号 |

**6点规则操作逻辑：**
- 如果一根棒的**棒体收盘超过6点**（ATR 5-10点环境）→ 机器判断为"方向性力量" → 预期继续走完全部10点摆动
- 如果棒体关闭在6点以内 → 仅为利润了结，方向未确认
- **11.75点阈值**：超过11.25-11.75点收盘 → 预示加速至15-20点

```
6点 = 5点标准scalp + 1点机器方向确认
  → 收盘>6 → 走向10点摆动
  → 收盘>11.75 → 走向15-20点
  → ATR若是8点 → 相应阈值提高（看back 4根棒的ATR）
```

### ▌熊旗三形态（Rose 讲解）

Rose 系统性整理了熊旗的三种类型：

```
类型1（传统旗形）：急跌 → 横盘整理 → 继续下跌
类型2（双底旗）：急跌 → 双底回到中点 → 再度突破下跌
类型3（尖峰-回撤-通道）：急跌 → 回撤 → 熊通道 → 回到回撤高点形成双顶 → 突破下跌
```

今日下午（K35后）：出现了类型3（尖峰-回撤-通道），Rose在双顶后做空。

- 突破三角形上轨 → 目标前高
- 突破三角形下轨 → 目标后续腿向下（约5626.50区域）
