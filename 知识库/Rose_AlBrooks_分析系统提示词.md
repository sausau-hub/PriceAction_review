## 一、本场 K 线逐棒解析（K2–K76）

> 视角：实时左侧，每棒收盘时的判断

### ▌开盘背景（Al Brooks 解说）
- **超大缺口高开**（今日高开约100点），大缺口增加趋势日概率，但也增加趋势不会走多远的可能
- 缺口方向略偏多，但当前位置远高于20日均线，不宜盲目追多
- 大概率场景：先走空，回到均线附近形成楔底/双底，再有效反弹；或者先走高形成楔顶再卖
- 月线视角：大概率未来数月走震荡区间（60%），继续新高（20%），继续新低（20%）

---

### ▌第一阶段：缺口高开后的空头趋势（K1–K27）

**K1（开盘棒）**
- 大阴棒，高开后立即卖出
- Al：远高于均线，大阴棒 = 大量卖单立即涌入，空头很可能在此直接做空
- 结论：这是空头信号棒，空头止损放高点，目标至少20点（10%日均波幅）

**K2–K6（初始空头通道）**
- 连续阴棒，依次创新低
- Al：这是典型的"缺口高开→趋势空头"结构，远高于均线，空头持续压制
- Rose：K6 的收盘是重要参考点，被套多头在此位置，后续会成为阻力

**K7–K11（小幅反弹/整理）**
- 几根小棒，多空胶着
- Rose：进入均线回归阶段，20棒均线仍在下方，均线会被市场"磁吸"
- Al：此处还不适合做多，要等到靠近均线的双底或楔底

**K12–K17（空头通道延续）**
- K12 低点是重要低点，空头开始在均线下方加速
- Rose：K16 是关键棒 — 阴棒收盘低于 K12 的开盘价，这意味着"小回调多头趋势"被彻底打破
- 判断：Always In Short（总持空）正式确立，连续阴棒实体重叠小

**K18–K20（均线附近的关键区域）**
- Al：20棒均线在此位置，市场来到了"平均价格"区域
- Rose：K18-20 形成 Always In Short 的三棒空头旗形
- 重要提示：此处是空头旗形，不是反转，空头继续

**K21–K24（空头旗形确认 + 新低）**
- K22 之后出现小双底形态，多头尝试反弹
- Rose：K25 是好的多头信号棒，但方向是逆空头通道，风险高
- Al：23 大阴棒收盘在低位但未破 K22 低点 — 空头力度稍弱，多头开始抵抗

**K25（关键多头信号棒）**
- 好的多头棒，高收盘
- Rose：信号棒是好的，但在紧密空头通道里做多是逆趋势，需要配合后续跟随棒
- K24+K25 棒体重叠达80% — 市场平衡，双方胶着

**K26–K27（三段下跌后的信号）**
- K26 强烈阴棒，K27 是不错的翻转棒（骑行棒/十字星类）
- Rose：这是三段下跌后的 27 号棒，有一定做多价值
- 目标：80点大波段的2R（即40点）已快到达
- Rose 分析：已经完成了40点目标（20点ATR的2倍），接下来可以加上10点 = 总共50–60点目标

### ▌第二阶段：底部形成 + 转折（K28–K36）

**K28（关键底部棒）**
- Rose：K28 是 Always In Short 末尾的关键底部，也是后续所有多空博弈的参考点
- 重要度：极高，后续分析中反复引用"K28 开盘/上三分之一/中点"作为目标和参考位
- 实际意义：K28 是这个下降段的"旗杆"底部，空头此后以它为基准量度目标

**K29–K32（小幅反弹 + 继续空头旗形）**
- 三棒横盘即为空头旗形
- Rose：K28–K30 组合 = 继续空头的组合，最终 K33 突破是空头的确认
- 判断：不是反转，是旗形延续，继续空

**K33（假突破 / 不好的空头棒）**
- Rose：空头本应在此大力推进，但 K33 是"弱收盘"的熊棒
- 关键判断：K33 没有收盘在低位，多头在此找到了支撑，空头失望
- 结果：多头被套的开始（买在 K33 close以上的空头被套）

**K34（突破棒）**
- 多头强棒，尝试突破
- Rose：任何买入 K34 以上的都是正确的，但注意空头也可能在 K28 上三分之一处等待反扑
- K34 成为后续多头行情的"旗杆"

**K35（回测/Low 1）**
- Rose：K35 是"Low 1"— 空头的第一次下探，通常不是好的做空点
- 重要概念：Low 1 之后多头还有机会，不急着卖
- K35 收盘在 K33 bad signal bar 区域 = 50%回撤完成

**K36（测试底部）**
- Rose：K36 回测了 K33 收盘价和 K32 低点，给前期空头一个平仓机会（break even）
- 判断：多头战场，Low 1已完成，下方买盘不远

**K37（多头关键棒）**

- Rose：买在 K34 的人现在得到了第二腿，这是有意义的多头信号

- 均线下方阴棒收盘
- Rose：空头看到"Low 2"机会，想要在此压制多头；但 K34 和 K37 的多头棒已经给出信号
- 判断：空头失去了最好的卖出时机，多头占优

**K39（失败的确认棒）**
- 多头本应在此确认突破，但 K39 未能收盘高于 K37 收盘价
- Rose：这是"失败"— 多头需要更多棒来证明
- 空头视角：35高、38高、40高 = 三重顶，空头在此激烈争夺

**K40–K51（震荡区间 + 第二次下探尝试）**
- 空头尝试形成 Low 2 双底和楔形

## 二、重要知识点（本场核心）

### 1. 缺口高开的四种结局（Al Brooks 教学）
```
大缺口高开后 → 四种结果概率相近：
① 缺口高开后先跌，到均线附近双底/楔底 → 反弹趋势 ✅ 本日走法
② 缺口高开横盘后形成双顶 → 大幅下跌
③ 缺口高开后小涨 → 形成楔顶 → 下跌
④ 缺口高开后横盘 → 最终方向不确定
核心：缺口越大，不会走的很远的概率越高
```

### 2. ATR 与目标设置（Rose 方法）
```
ATR约14点 → 
  Scalp（短线）= 20点
  Swing 1R = 40-41点（需走41点才给你40点）
  Swing 2R = 60点
  3R/更大 = 80点+（关缺口目标）
记忆公式：scalp ≈ 10%日均波幅，swing ≈ 40%日均波幅
```

### 3. Bear Flag 三要素（Rose 判断标准）
```
① Always In Short 背景（2-3根大阴棒收低位，实体重叠小）
② 出现3棒以内横盘 = 旗形
③ 下一棒必须关闭阴棒 → 旗形突破确认
进场：旗形下方做空；止损：旗形高点；目标：旗杆2倍
```

### 4. Always In Short / Long 判断
```
AIS 确立条件（空头）：
- 2-3根连续阴棒
- 收盘接近低点（下三分之一）
- 实体之间重叠部分 < 50%
- 在均线下方
AIL 确立条件（多头）：
- 2根连续多头棒从均线下方穿越均线
- 收盘在上三分之一
- 此后 pullback 仍是 買点
```

### 5. Low 1 / High 1 不做原则
```
Low 1 = 趋势中第一次回调低点，"不是好的做空点"
High 1 = 趋势中第一次回调高点，"不是好的做多点"
原因：第一次回调后，趋势方向力量仍未耗尽，止损难放
正确做法：等 Low 2 / High 2 再进场
```

### 6. 棒体重叠分析
```
80%+ 棒体重叠 → 市场平衡，双方胶着 → 等待突破方向
< 50% 重叠 → 趋势明确，跟随趋势
关键棒体参考点（Rose 独特方法）：
- 上三分之一：卖出/阻力区
- 中点（50%）：测试/平衡点
- 下三分之一：买入/支撑区
```

### 7. 信号棒质量判断（Rose 版）
```
好的做多信号棒 = 收盘在高位（上三分之一）+ 棒体不太大
好的做空信号棒 = 收盘在低位（下三分之一）+ 棒体不太大
差的信号棒 = 收盘在中间（十字星）→ 等待下一棒确认
大棒信号棒 → 等跟随棒收盘确认再进，不追高/低
```

### 8. 目标计算 —— 旗形章节（Rose 引用 Al Brooks 书）
```
来源：Al Brooks "Bar by Bar Price Action" 旗形章节 P.39（蓝书）
规则：
- 做空标准目标 = 旗杆高度 × 1（1倍）
- 做空延伸目标 = 旗杆高度 × 2
- 止损放在旗形最高点上方
- 平半仓在1R，剩余留runner
```

### 9. 均线的磁吸效应（Al Brooks 核心思想）
```
当价格远高于20棒均线 → 不适合买
当价格回到均线附近 → 买入兴趣出现
当价格远低于均线 → 不适合卖
磁吸规律：市场总会回到均线，然后再决定方向
实用推论：开盘初期远离均线 → 要么横盘等均线追上来，要么价格回调
```

### 10. 概率思维 —— Al Brooks 核心
```
Al 的口头禅：
"我说60%概率，不是100%"
"每一tick，概率都在变"
"越接近某个关键位，走到该位的概率越高"
应用：不用预测方向，用概率管理仓位和止损
  → 在概率高位（70%+）时才全仓进场
  → 50-50 的时候，要么不进，要么极小仓
```

## 三、系统提示词（用于复盘分析）

```
你是一位专业的ES期货价格行为分析师，深入研习 Al Brooks 和 Rose 的分析风格。

━━━━━━━━━━━━━━━━━━━━━━
【你的分析框架】
━━━━━━━━━━━━━━━━━━━━━━

**Al Brooks 视角**（宏观 + 逻辑框架）：
- 先判断今日背景：缺口大小、均线方向、前日结构
- 说明当前所处结构：趋势 / 震荡区间 / 通道
- 用概率语言：「60%可能...，40%可能...」
- 分析每根 K 线的含义：多空双方在争什么？谁在买？谁在卖？为什么？
- 重点分析：棒体位置（高/中/低三分之一）、是否收在低位/高位
- 均线位置的重要性：远高于均线不买，远低于均线不卖

**Rose 视角**（实战 + 量化判断）：
- 建立 20 棒均线区间范围（20 bar range）
- 识别 Always In Short / Always In Long 信号
- 量化目标：ATR × 倍数（1R=2×ATR，2R=4×ATR）
- 识别 Flag 结构：旗杆 + 旗形 = 目标 = 旗杆高度
- 追踪关键参考棒：开盘棒、最强棒、重要信号棒的「上/中/下三分之一」
- 判断 Low 1 / Low 2 / High 1 / High 2 的做空/多价值

━━━━━━━━━━━━━━━━━━━━━━
【逐棒解读规则】
━━━━━━━━━━━━━━━━━━━━━━

1. **全覆盖**：每根 K 线都要提到，关键棒单独详解，非关键棒可分组
2. **实时视角**：写作视角是「这根 K 线刚收盘，我正在看左侧」
   ✅ 正确：「K33 刚收盘，收盘在中间，这让空头很失望...」
   ❌ 禁止：「K33 之后市场开始反弹...」（事后视角）
3. **多空博弈语言**：每根棒说明「多头在做什么」「空头在做什么」「谁占优」
4. **引用关键参考位**：
   - 当前K线与前方重要棒的关系（开盘价、收盘价、上/中/下三分之一）
   - 与 20棒均线的位置关系
   - 与前日收盘、重要支撑阻力的关系

━━━━━━━━━━━━━━━━━━━━━━
【判断结构的必答题】
━━━━━━━━━━━━━━━━━━━━━━

分析时必须回答：
1. 今日是哪种日型？（趋势日 / 震荡区间日 / 通道日 / 缺口反转日）
2. 当前 Always In 方向是什么？（多 / 空 / 不明确）
3. 有无 Flag 结构？旗杆在哪？目标是多少？
4. 当前概率是多少？（用 Al Brooks 式概率语言）
5. 下一步等什么信号？（具体：等某棒收盘在哪个位置）

━━━━━━━━━━━━━━━━━━━━━━
【目标设置模板（Rose 方法）】
━━━━━━━━━━━━━━━━━━━━━━

背景信息需要：ATR 值（通常 ES 5分钟约 10–20 点）

计算：
  Scalp 目标 = 1× ATR（最小）
  1R Swing = 2× ATR（标准波段）
  2R Extended = 4× ATR
  旗形目标 = 旗杆高度（从旗杆顶部/底部量）
  双重目标 = 旗杆高度 × 2

平仓规则：
  → 达到 1R 时，平掉半仓
  → 剩余留 runner，移动止损至成本价
  → 达到 2R，再平半仓
  → 极端情况可以追求 3R–4R（留最小仓位）

━━━━━━━━━━━━━━━━━━━━━━
【进场标准（简版）】
━━━━━━━━━━━━━━━━━━━━━━

做多条件（优先级排列）：
① Always In Long 确立（2根从均线下方穿越的连续多头棒）+ 收盘在上三分之一
② 三段下跌后的楔底/双底 + 好的多头信号棒（收在高位）
③ Bull Flag 突破（旗形结束后第一根大阳棒）
④ Low 2 位置 + 好的信号棒

做空条件（优先级排列）：
① Always In Short 确立 + 连续收在下三分之一
② Bear Flag 突破（3棒横盘后第一根大阴棒）
③ High 2 位置 + 好的信号棒
④ 楔顶/三重顶 + 阴棒确认

━━━━━━━━━━━━━━━━━━━━━━
【禁止行为】
━━━━━━━━━━━━━━━━━━━━━━
🚫 在 Low 1 / High 1 逆趋势进场
🚫 缺口高开且远高于均线时盲目做多
🚫 在震荡区间中间位置进场（只在边缘进）
🚫 信号棒太大时追进（等下一棒）
🚫 不说概率，只说"一定会"
🚫 事后视角解读（"这根K线之后..."）
```
